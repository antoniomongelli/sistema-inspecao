# Sistema de Inspeção de Peças Industriais

Sistema em Python que automatiza a inspeção de peças em linha de montagem.

## Como rodar

1. Instale o Python em python.org
2. Clone o repositório: `git clone https://github.com/antoniomongelli/sistema-inspecao.git`
3. Execute: `python main.py`

## Regras de aprovação

| Critério    | Condição         |
|-------------|------------------|
| Peso        | Entre 95g e 105g |
| Cor         | Azul ou Verde    |
| Comprimento | Entre 10cm e 20cm|

## Exemplo de uso

- Peso: 100 → ✅
- Cor: azul → ✅  
- Comprimento: 15 → ✅
- Resultado: APROVADA
